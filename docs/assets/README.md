# Documentation assets

Place optional screenshots / exported diagrams here:

- **`images/`** — UI captures  
  - [`installation/`](images/installation/) — HACS / add integration  
  - [`configuration/`](images/configuration/) — config flow  
  - [`dashboard/`](images/dashboard/) — Lovelace  
  - [`troubleshooting/`](images/troubleshooting/) — logs / repairs  
- **`diagrams/`**  
  - [`sources/`](diagrams/sources/) — editable sources (draw.io, etc.)  
  - [`exports/`](diagrams/exports/) — PNG/SVG exports  
- **`videos/`** — [Links only](videos/README.md) (no large binaries in git)

Markdown docs prefer **relative links** into this folder when assets exist.

Mermaid diagrams also live under [`architecture/diagrams/`](../architecture/diagrams/).
