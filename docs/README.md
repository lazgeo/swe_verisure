# Documentation (`docs/`)

This directory contains **production-oriented documentation** for the My Verisure Home Assistant integration: user guides, developer guides, architecture notes, API reference, runnable examples, and a roadmap.

## Layout

| Directory | Purpose |
|-----------|---------|
| [user-guide/](user-guide/README.md) | Installation, configuration, entities, services, usage |
| [developer-guide/](developer-guide/README.md) | Local dev, testing, modules, patterns |
| [architecture/](architecture/README.md) | System design, data flow, diagrams |
| [api-reference/](api-reference/README.md) | Per-entity and per-service reference |
| [examples/](examples/README.md) | Automation, script, and dashboard YAML |
| [technical/](technical/README.md) | GraphQL, auth, session, models |
| [integration/](integration/README.md) | External platforms (Node-RED, assistants) |
| [roadmap/](roadmap/README.md) | Gaps, improvements, phases |
| [assets/](assets/README.md) | Images and diagram sources (optional) |

## Entry points

- **Site hub:** [index.md](index.md)  
- **Users:** [user-guide/getting-started.md](user-guide/getting-started.md)  
- **Contributors:** [developer-guide/setup.md](developer-guide/setup.md)
